## String Split

creating a Implementation of Standard library's str::split(), which returns succesive substrings.

### Example

```rust
StringSplit::new("Hello world, Here I am", ",")
/// Output: [Hello world, Here I am]
```

### Local Development

1. clone the repo `bash git clone https://github.com/subhadeep-sengupta/string-split.git`

2. cd into the repo

3. Build the project `bash cargo build`
   For tests `bash cargo test`

For further development feel free to open a PR!
